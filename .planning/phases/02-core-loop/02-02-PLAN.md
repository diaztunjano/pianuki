  <action>
Create `src/game/gameLoop.ts` — a pure module (no React) containing the game update logic. Import `useBoundStore` from `../stores`, `isNoteMatch` from `./noteMatch`, and `Enemy` from `./enemyTypes`.

**Export `update(dt: number): void`** — called each fixed timestep tick (dt in ms). Reads state via `useBoundStore.getState()` at the top of the function (not cached). Performs the following steps in order:

1. **Spawn enemies:** Read `spawnQueue`, `spawnAccumulator`, `spawnIntervalMs`. If `spawnQueue.length > 0`, add `dt` to accumulator. If accumulator >= spawnIntervalMs, pop the first entry from spawnQueue, call `spawnEnemy(entry)`, reset accumulator to 0. Update `spawnAccumulator` in the store via `useBoundStore.setState({ spawnAccumulator: newValue })`.

2. **Move enemies:** For each enemy with `state === 'alive'`, advance `pathT += enemy.speed * dt`. Use immer-compatible mutation: `useBoundStore.setState((s) => { for (const e of s.enemies) { if (e.state === 'alive') e.pathT += e.speed * dt } })`.

3. **Check goal reached:** For each enemy with `pathT >= 1.0 && state === 'alive'`, call `useBoundStore.getState().enemyReachedGoal(enemy.id)`. Check enemies from the store AFTER the move step.

4. **Check note matches:** Read `activeNotes` from store. For each enemy with `state === 'alive'`, call `isNoteMatch(enemy, activeNotes)`. If matched: call `damageEnemy(enemy.id, 1)`.

5. **Wrong note detection:** Track last checked NoteOn event. Read `events` from store. Find any new `NoteOn` events since the last checked timestamp (track via module-level `let lastCheckedEventTs = 0`). For each new NoteOn, check if its `note` matches ANY alive enemy's `targetNote`. If no match: call `triggerWrongNote()`. Update `lastCheckedEventTs` to the latest event timestamp.

6. **Tick dying enemies:** Call `useBoundStore.getState().tickDyingEnemies()`.

7. **Remove dead enemies:** Call `useBoundStore.getState().removeDeadEnemies()`.

8. **Tick wrong note flash:** Call `useBoundStore.getState().tickWrongNoteFlash()`.

9. **Check wave complete:** If `spawnQueue.length === 0` AND all enemies are gone (enemies array is empty after dead removal): check if more waves remain. If yes: set `gamePhase: 'wave-clear'` via `useBoundStore.setState({ gamePhase: 'wave-clear' })`. If no more waves: set `gamePhase: 'idle'` (level complete).

**Important anti-patterns to avoid:**
- Do NOT cache `useBoundStore.getState()` at function start and reuse — state changes between steps. Read fresh state for each step that needs it.
- Do NOT use `setTimeout` for spawning — use the accumulator pattern.
- Do NOT check wrong note on every frame — only on new NoteOn events.

Replace the current simple `loop` function with a fixed-timestep accumulator pattern:
```
const TIMESTEP = 1000 / 60  // ~16.67ms
let previousTimeMs = 0
let accumulator = 0

const loop = (currentTimeMs: number) => {
  const raw = currentTimeMs - previousTimeMs
  const deltaTimeMs = Math.min(raw, 100) // clamp: prevent spiral of death
  previousTimeMs = currentTimeMs

  const { gamePhase } = useBoundStore.getState()

  if (gamePhase === 'playing') {
    accumulator += deltaTimeMs
    while (accumulator >= TIMESTEP) {
      update(TIMESTEP)
      accumulator -= TIMESTEP
    }
  }

  drawFrame(ctx, canvas.width, canvas.height)
  frameId = requestAnimationFrame(loop)
}
```

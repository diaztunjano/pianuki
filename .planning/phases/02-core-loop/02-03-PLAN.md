```ts
const gamePhase = useBoundStore((s) => s.gamePhase)
const startGame = useBoundStore((s) => s.startGame)

const advanceWave = useBoundStore((s) => s.advanceWave)
```

All overlay screens share the same wrapper: `absolute inset-0 flex items-center justify-center bg-black/70 z-20`. Use Tailwind for all styling. Each screen is a centered card with semi-transparent backdrop.

**2. Add ESC key listener for pause/resume:**
Add a `useEffect` with a `keydown` event listener on `window`:
```ts
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    if (e.key === 'Escape' && !e.repeat) {
      const { gamePhase, pauseGame, resumeGame } = useBoundStore.getState()
      if (gamePhase === 'playing') pauseGame()
      else if (gamePhase === 'paused') resumeGame()
    }
  }
  window.addEventListener('keydown', handleKeyDown)
  return () => window.removeEventListener('keydown', handleKeyDown)
}, [])
```
Import `useBoundStore` from `../stores` (use `getState()` in the handler, not hooks — the handler is not a render function).

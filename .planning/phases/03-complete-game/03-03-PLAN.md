  <name>Task 1: Build LevelSelectScreen with path/map layout</name>
  <files>src/components/LevelSelectScreen.tsx</files>
  <action>
Create the level select screen as a React component. This replaces the stub placeholder created in Plan 02's AppShell.

**Layout** (per user decisions — path/map style like Mario World):
- Vertical scrollable container with level nodes arranged in a zigzag pattern
- SVG path in the background connecting nodes with a bezier curve
- Each node is a circular button showing level name, star rating (1-3 stars), and note range preview
- Locked nodes are grayed out with opacity-50 and show a lock icon (use "🔒" or a padlock SVG)
- Level 0 is always unlocked; subsequent levels require `progress.levels[i-1].stars >= 1`

**Node positions** — zigzag pattern:
```
const NODE_POSITIONS = [
  { x: '25%', y: 80 },   // Level 0 — left
  { x: '50%', y: 180 },  // Level 1 — center
  { x: '75%', y: 280 },  // Level 2 — right
  { x: '50%', y: 380 },  // Level 3 — center
  { x: '25%', y: 480 },  // Level 4 — left
]
```

**Each level node shows:**
- Level name from `LEVEL_CONFIGS[i].name` (e.g. "Meadow", "Forest")
- Star rating: filled stars for earned stars, empty/dim stars for unearned. If 0 stars (not completed), show note range label instead.
- Best score (bestAccuracy %) if completed
- Note range label from `LEVEL_CONFIGS[i].noteRangeLabel`

**SVG connecting path:**
- An SVG `<path>` element connecting the node positions with smooth curves
- Stroke color: `#ffffff15` (subtle), strokeWidth: 3, fill: none
- Use quadratic bezier curves between adjacent node positions

**Navigation buttons:**
- "Stats" button in top-right corner → `navigateTo('stats')`
- "Settings" button (gear icon or text) in top-left corner → opens settings (Plan 04 will add the real panel; for now, just reserve the spot)
- Title "PIANUKI" at the top center

**Selectors** — use narrow selectors to avoid unnecessary re-renders:
```tsx
const progress = useBoundStore((s) => s.progress)
const startLevel = useBoundStore((s) => s.startLevel)
const navigateTo = useBoundStore((s) => s.navigateTo)
```

**Import** LEVEL_CONFIGS from waveConfig for level metadata (name, noteRangeLabel, etc.)

**Update AppShell.tsx**: Replace the inline stub `LevelSelectScreen` with an import from `./LevelSelectScreen`. Delete the stub function.
